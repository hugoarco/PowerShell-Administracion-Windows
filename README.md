# Script-Administracion-Programas-Powershell
Script desarrollado en PowerShell orientado a la administración básica de equipos Windows. El proyecto permite gestionar programas mediante un menú interactivo, incluyendo funciones para instalar, desinstalar y comprobar aplicaciones utilizando Winget.

También incorpora un sistema básico de identificación de usuarios, control de opciones y manejo de errores para mejorar la experiencia de uso. El objetivo del proyecto es practicar automatización de tareas, administración de software y creación de herramientas sencillas para facilitar la gestión de equipos informáticos.
